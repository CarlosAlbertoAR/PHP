VSCode

 - Install PHP extension Devsense

 - C:\Users\_UserName_\AppData\Roaming\Code\User\settings.json
 
   Add: "php.executablePath": "C:\\xampp\\php\\php.exe"

 - To Debug press Ctrl Shift D and select option "Launch built-in server and debug"

PHP

 - Add php dir to "Enviroment variables" in Path
   Ex: C:\xampp\php

 https://docs.devsense.com/en/vscode/debug/xdebug-win

 - Download PHP XDebug in : https://xdebug.org/download#releases

 - Save on php ext dir:

   Ex: C:\xampp\php\ext\php_xdebug-3.3.2-8.2-vs16-x86_64.dll


 - Edit PHP.ini

 - Enable PHP extensions: xsl, mbstring

 - Add section:

[XDEBUG]
zend_extension="C:\xampp\php\ext\php_xdebug-3.3.2-8.2-vs16-x86_64.dll"
xdebug.mode=debug
xdebug.client_host = 127.0.0.1
xdebug.client_port = 9003
xdebug.start_with_request=yes

Verify Xdebug
